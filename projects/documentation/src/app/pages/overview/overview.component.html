Shoelace - библиотека, основанная на web компонентах. shoelace-style-angular добавляет поддежку свойств (Property binding) и событий (Event binding) в шаблонах angular, которые можно использовать так, как если бы это были компоненты angular.

Рассмотрим простой пример открытия диалогового окна. У элемента sl-dialog предусмотрено свойство open, для его открытия. Для управления этим свойством воспользуемся двусторонней привязкой, чтобы можно было с одной стороной устанавливать свойство в коде, например, для открытия. Но и значение isDialogOpen сохраняло актуальное состояние, например, при закрытии окна через крестик.
